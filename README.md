## 📄 דוח הסבר – משחק “תפוס חרקים” (JavaScript)

### 🎮 תיאור קצר של המשחק

המשחק “תפוס חרקים” הוא משחק אינטראקטיבי בדפדפן, שבו השחקן בוחר חרק מועדף ולאחר מכן צריך **לתפוס כמה שיותר חרקים בלחיצות עכבר בזמן אמת**.
המשחק כולל מסכי מעבר, מדידת זמן, ניקוד, הופעת אובייקטים רנדומליים והודעות משוב לשחקן.

---

## 🎯 מטרות למידה בקורס

באמצעות בניית המשחק התלמידים לומדים:

* איך JavaScript שולט בהתנהגות של דף אינטרנט
* איך יוצרים אינטראקציה עם המשתמש
* איך מפתחים לוגיקה של משחק אמיתי
* איך מחברים HTML + CSS + JS למערכת אחת עובדת

---

## 🧠 מה לומדים ב-JavaScript בפרויקט הזה

### 1️⃣ עבודה עם ה-DOM

התלמידים לומדים:

* לבחור אלמנטים מהעמוד (`querySelector`, `getElementById`)
* לשנות תוכן דינמי (`innerHTML`)
* להוסיף ולהסיר אלמנטים מהמסך (`appendChild`, `remove`)

📌 דוגמה:

```js
const game_container = document.getElementById('game-container')
```

---

### 2️⃣ טיפול באירועים (Events)

המשחק מבוסס על **פעולות של המשתמש**:

* לחיצה על כפתור התחלה
* בחירת חרק
* לחיצה על חרק במשחק

📌 התלמידים מבינים:

* מה זה `addEventListener`
* איך פעולה של המשתמש מפעילה פונקציה

```js
btn.addEventListener('click', () => {
  screens[1].classList.add('up')
})
```

---

### 3️⃣ פונקציות ולוגיקה של משחק

JavaScript אחראי על:

* יצירת חרקים
* תפיסת חרק
* חישוב ניקוד
* בדיקת תנאים (אם עברנו ציון מסוים)

📌 דוגמה:

```js
function catchInsect() {
  increaseScore()
  this.classList.add('caught')
}
```

---

### 4️⃣ עבודה עם משתנים ומצב (State)

התלמידים לומדים לשמור מידע:

* זמן שחלף (`seconds`)
* ניקוד (`score`)
* החרק שנבחר (`selected_insect`)

📌 זהו בסיס חשוב להבנת:

* איך משחק “זוכר” מה קרה
* איך נתונים משתנים לאורך זמן

---

### 5️⃣ עבודה עם זמן (Timers)

המשחק משתמש ב-JavaScript כדי:

* למדוד זמן משחק
* ליצור חרקים בהפרשי זמן

📌 מושגים נלמדים:

* `setInterval`
* `setTimeout`

```js
setInterval(increaseTime, 1000)
```

---

### 6️⃣ רנדומליות (Random)

JavaScript יוצר חוויה לא צפויה:

* כל חרק מופיע במקום אחר
* כל חרק מסתובב בזווית שונה

📌 נלמד:

```js
Math.random()
```

---

### 7️⃣ עבודה עם תנאים (if)

התלמידים לומדים:

* לבדוק מתי להציג הודעה
* להגיב לניקוד גבוה

```js
if(score > 19) {
  message.classList.add('visible')
}
```
